Projects

Ph.D. Research Project

My doctoral research is focussed on development of photovoltaic device optimization models by combining simulation tools, experimental data and machine learning. These optimization models will improve performance of silicon based solar cells as well as optimize their processing techniques.

XSolar-Hetero Simulation Platform

https://xsolar-hetero.sg (June 2016 - Ongoing)

This project aims at delivering a publicly accessible website where researchers can create solar cell architectures based on material libraries available on the website, perform device and process simulations online, store their work on a database, upload measurement data from characterization tools, compare this data with simulated data and use machine learning models to optimize the cell design and its fabrication.

Fabrication of N- Channel Enhancement type MOSFET

CHTM (Sep 2019 - Dec 2019)

NMOS Devices were fabricated for an academic project on a 525 um thick p type wafer. The processing steps involved in fabrication were oxidation, photo-lithography, diffusion, etching and mettalization. Device was characterized at each step and fabricated device has ideal I-V property.

Simulation API development

Wolfram Tech - 2018 (March 2017 - Dec 2018)

Desktop based device and process photovoltaic simulation programs were converted to form based REST API's with JSON output, that were used as microservices in the XSolar-Hetero project. These simulation models were either built in tools like PC1D, Quokka, Sentaurus or were written in programming languages like C++ / Matlab / Mathematica.

Remotely programmable smart thermostat

UNM (Jun 2020 - Jul 2020)

In this academic project linux executable code was developed and deployed on a buildroot based linux image, monitoring, and reporting temperature collected via a thermostat to an AWS EC2 instance. Thermostat status was reported via HTTP Post method, and GET requests were used for remote config.

Weather Reporting station prototype

During my internship at National university of Singapore a low cost, low power, portable, power efficient weather reporting station was built based on open source hardware and software (Raspberry pi, Arduino board and embedded C), the device was remotely configurable and updated weather data to a labview dashboard.
